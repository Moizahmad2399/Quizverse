## 📖 About The Project

**QuizVerse** is a fully responsive, interactive quiz web application built using pure client-side technologies — HTML5, CSS3, and Vanilla JavaScript. The application challenges users across five knowledge domains with difficulty-based question pools, real-time countdown timers, dynamic answer feedback, and a detailed results report that can be downloaded as a PDF.

The project was designed with a **cosmic dark/light theme**, glassmorphism UI, and smooth CSS animations to deliver a premium, production-quality user experience — all without any front-end frameworks.

---

## ✨ Features

- 🧠 **5 Knowledge Categories** — Science, History, Technology, Mathematics, Geography
- 🎯 **3 Difficulty Levels** — Easy (30s), Medium (20s), Hard (12s) with unique question pools
- ⏱️ **Animated Countdown Timer** — SVG ring that changes color as time runs out
- 📊 **Multi-colored Segmented Progress Bar** — tracks correct, wrong, and current question
- ✅ **Dynamic Answer Highlighting** — instant green/red feedback with explanations
- 🌙 **Dark / Light Mode Toggle** — full theme switch using CSS custom properties
- 📄 **Download Results as PDF** — styled report generated client-side using jsPDF
- 🔀 **Randomized Questions** — shuffled pool every game for replayability
- 📱 **Fully Responsive** — seamless experience on mobile, tablet, and desktop
- 🔙 **Back Button** — return to home screen anytime during the quiz

## 🎮 How To Play

1. **Select a Category** — choose from Science, History, Technology, Math, or Geography
2. **Choose Difficulty** — Easy, Medium, or Hard
3. **Click Launch Quiz** — answer 5 randomized questions before the timer runs out
4. **Get Instant Feedback** — see correct/wrong highlights and read the explanation
5. **View Your Results** — check your score, stats, and full answer review
6. **Download PDF** — save your results report
